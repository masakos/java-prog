## 問題2
以下の成績判定プログラムを以下のように改良してください。

- judge() と addBonus() を ScoreUtil クラスへ移動してください。
- isPerfect(int score) メソッドをScoreUtilクラスに追加し、100点なら true、それ以外は false を返すようにしなさい。
- main() を修正して、満点かどうかも表示しなさい。

```java
public class ScoreApp {

    public static void main(String[] args) {

        int score = 75;

        System.out.println("判定：" + judge(score));
        System.out.println("5点加算：" + addBonus(score));

    }

    public static String judge(int score) {
        if (score >= 60) {
            return "合格";
        } else {
            return "不合格";
        }
    }

    public static int addBonus(int score) {
        return score + 5;
    }
}
```

## 問題3：プログラムを一時停止する

次のプログラムは、数字を2秒ごとに表示したいプログラムです。しかし、現在は一瞬で表示されてしまいます。

Java APIで **`java.lang.Thread`** クラスを調べ、一時停止するメソッドを利用して、**1秒ごとに表示されるように修正してください。

```java
public class Main {
    public static void main(String[] args) {
        for (int i = 1; i <= 5; i++) {
            System.out.println(i);
        }
    }
}
```
