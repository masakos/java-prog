## 問題2
成績判定プログラムを以下のように改良してください。

- judge() と addBonus() を ScoreUtil クラスへ移動しなさい。
- isPerfect(int score) メソッドを追加し、100点なら true、それ以外は false を返すようにしなさい。
- main() を修正して、満点かどうかも表示しなさい。

```java
public class ScoreApp {

    public static void main(String[] args) {

        int score = 75;

        System.out.println("判定：" + judge(score));
        System.out.println("5点加算：" + addBonus(score));

    }

    public static String judge(int score) {
        if (score >= 60) {
            return "合格";
        } else {
            return "不合格";
        }
    }

    public static int addBonus(int score) {
        return score + 5;
    }
}
```
