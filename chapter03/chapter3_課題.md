
## 提出された課題で気づいたこと
- ソースファイル名は、「クラス.java」にする
- クラス名はアルファベット大文字で始める
- 自動フォーマット機能を使おう
  - Ctr + ,  ->  format on save
- ソースコード中にリテラルの文字列以外で全角文字を入れない
- なるべくIDEの補完を使用するとミスが減る
- {} かっこが正しくブロックごとに閉じているか確認する
  - { または } の横にカーソルを置くと対応する括弧が自動でハイライトされる
  - `Ctrl + Shift + \`  で対応する {} にカーソルが移動する
- ガイドを消したいとき
  - Ctr + ,  -> inlay hints 









# Java課題（if文・条件式）

それぞれ、Javaファイルを作成して、以下のプログラムを提出してください。

## 提出方法
- 各問題ごとに 指定された名前の `.java` ファイルを作成する
- 作成したファイルを1つのフォルダにまとめる 
    - (1)課題提出用のフォルダを作成する
        - C:\Users\(それぞれのユーザー名）\java課題提出用\0518_2610xx 　（今日の日付_学籍番号）
    - (2)作成した `.java` ファイルを上記のフォルダに入れる
- フォルダを ZIP 圧縮する
    - フォルダを右クリック -> 圧縮先 -> zipファイル
- Teams の課題に添付して提出する

※ 終わったものだけでよいので提出してください。(途中ものもは途中まででＯＫ)
