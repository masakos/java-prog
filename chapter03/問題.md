## switch文問題

【問題1】
次のプログラムを実行したとき、画面に何が表示されますか？

```java
int rank = 2;

switch (rank) {
    case 1:
        System.out.println("金メダル");
        break;
    case 2:
        System.out.println("銀メダル");
        break;
    case 3:
        System.out.println("銅メダル");
        break;
    default:
        System.out.println("参加賞");
        break;
}
```


【問題2】
次のプログラムを実行したとき、画面に表示されるものをすべて答えなさい。

```java
int signal = 1;

switch (signal) {
    case 1:
        System.out.println("進め");
    case 2:
        System.out.println("止まれ");
        break;
    case 3:

        break;

```

【問題3】
次のプログラムを実行したとき、画面に何が表示されるか答えなさい。

```java
String weather = "槍（やり）";

switch (weather) {
    case "晴れ" -> System.out.println("洗濯をする");
    case "雨" -> System.out.println("傘を持っていく");
    default -> System.out.println("自宅待機します");
}
```

【問題4】
次の if 文のプログラムを、switch 文を使って同じ動きをするように書き換えなさい。

```java
int month = 12;

if (month == 12) {
    System.out.println("年末です");
} else if (month == 1) {
    System.out.println("年始です");
} else {
    System.out.println("普通の月です");
}
```


## while文

- 次のJavaコードを実行したとき、画面に「Hello」は何回表示されますか？

```java
int count = 0;
while (count < 3) {
    System.out.println("Hello");
    count++;
}
```

- 次のコードを実行したとき、どのような結果になりますか？
```java
int i = 5;
while (i > 0) {
    System.out.println(i);
    // iの更新処理がない
}
```
