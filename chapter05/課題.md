# 課題11: コマンドライン引数を利用した平均点の計算
コマンドライン引数として渡された点数の平均点を求めて表示するプログラムを作成しなさい。

### 条件

- コマンドライン引数には、0～100の整数の点数が1つ以上渡されるものとする。
- コマンドライン引数は `Integer.parseInt()` を使用して整数に変換すること。
- 平均点は小数点以下も表示すること。

### 実行例

```text
java ScoreCalculator 80 75 92 68
```

### 実行結果

```text
平均点：78.75
```

- ヒント
```
Integer.parseInt() は、
文字列（String）を整数（int）に変換するメソッド です。

コマンドライン引数 args はすべて文字列として渡されるため、数値として計算するには Integer.parseInt() を使って変換する必要があります。

基本的な使い方
int num = Integer.parseInt("100");

System.out.println(num);
```
