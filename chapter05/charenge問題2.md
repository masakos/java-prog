## challenge2_1：素数判定

### 問題

整数を受け取り、その数が**素数**であるかどうかを判定するメソッドを作成してください。

素数とは、**1より大きく、自分自身と1以外では割り切れない数**のことです。

### 仕様

| 項目 | 内容 |
|------|------|
| メソッド名 | isPrime |
| 戻り値 | boolean |
| 引数 | int number |
| 処理 | 素数なら `true`、それ以外なら `false` を返す |

### 実行例

```java
System.out.println(isPrime(7));
System.out.println(isPrime(10));
```

出力

```
true
false
```

## challenge2_2：回文判定

### 問題

文字列を受け取り、その文字列が**回文**かどうかを判定するメソッドを作成してください。

回文とは、**前から読んでも後ろから読んでも同じ文字列**のことです。

### 仕様

| 項目 | 内容 |
|------|------|
| メソッド名 | isPalindrome |
| 戻り値 | boolean |
| 引数 | String text |
| 処理 | 回文なら `true`、それ以外なら `false` を返す |

### 実行例

```java
System.out.println(isPalindrome("level"));
System.out.println(isPalindrome("java"));
```

出力

```
true
false
```

## challenge2_3：最大公約数

### 問題

2つの整数を受け取り、その**最大公約数**を返すメソッドを作成してください。

※この問題では、まだユークリッドの互除法は使用せず、総当たりで求めてください。

### 仕様

| 項目 | 内容 |
|------|------|
| メソッド名 | gcd |
| 戻り値 | int |
| 引数 | int a, int b |
| 処理 | 最大公約数を返す |

### 実行例

```java
System.out.println(gcd(24, 36));
```

出力

```
12
```

## challenge2_5：ランレングス圧縮

### 問題

- 同じ文字が連続している部分を、「文字 + 出現回数」の形式に変換するメソッドを作成してください。

## ランレングス圧縮(Run-Length Encoding：RLE)とは、
- 同じ文字が連続している部分を、「文字」と「連続回数」で表現する圧縮方法です。
- 画像データやFAXなどでも使われる、とても基本的な圧縮アルゴリズムです。

### 仕様

| 項目 | 内容 |
|------|------|
| メソッド名 | compress |
| 戻り値 | String |
| 引数 | String text |
| 処理 | ランレングス圧縮した文字列を返す |

### 実行例

```java
System.out.println(compress("AAAABBBCCAAA"));
```

出力

```
A4B3C2A3
```


## challenge2_8：ユークリッドの互除法

### 問題

- 2つの整数を受け取り、**ユークリッドの互除法**を用いて最大公約数を求めるメソッドを作成してください。
- ユークリッド互除法のロジック自体は、WEBなどで調べてください。

#### ユークリッドの互除法（Euclidean Algorithm）とは、
- 2つの整数の最大公約数（GCD）を、非常に効率よく求めるアルゴリズムです。
- 約2300年前に、古代ギリシャの数学者 ユークリッド が考えた方法で、現在でもコンピュータで広く使われています。

### 仕様

| 項目 | 内容 |
|------|------|
| メソッド名 | gcdEuclid |
| 戻り値 | int |
| 引数 | int a, int b |
| 処理 | ユークリッドの互除法を用いて最大公約数を返す |

### 実行例

```java
System.out.println(gcdEuclid(24, 36));
System.out.println(gcdEuclid(56, 42));
```

出力

```
12
14
```
