
## メソッド

#### メメソッの定義
```
public static void メッソド名() {
    メソッドが呼び出されたときの処理
}
```

#### メソッドの呼び出し
```
メソッド名()
```

### 引数（ひきすう：argument）
- 呼びだし元から値を渡すことができる
- 呼び出されたメソッド側では、渡される値を受け取って処理に利用できる
- 引数は、カンマ(,) で区切って、複数渡すことができる

#### 引数を利用したメソッドの定義
```java
public static void メソッド名(引数リスト){
    メソッドの処理
}
```

#### 引数を利用したメソッドの呼び出し
```java
メソッド名(引数リスト)
```

### 戻り値
#### 戻り値を利用したメソッドの定義
```java
public static 戻り値の型　メソッド名(引数リスト){
    メソッドの処理
    return 戻り値;
}
```

- 戻り値の型 `void` は何も返さないという意味
- 戻り値は1つしか使えない
- 戻り値には、`retrun 100` や `retrun "hello"`  のようリテラルを指定することもできる
- return文は値を戻すだけでなく、メソッドの終了も行う


#### 戻り値を利用したメソッド呼び出し
```java
型 変数名 = メソッド名(引数リスト);
```




## オーバーロード
- 同じ名前のメソッドを定義することを `オーバーロード` という
- 仮引数の `個数` か `型` が異なれば、同じ名前のメソッドを複数定義できる
